# Windows Fundamentals

This repository contains hands-on exercises and notes for **Microsoft Windows Fundamentals**.  
It documents Level 1 tasks including user account management, password handling, and User Account Control (UAC).

## Repository Structure

- **Level-1-Basics/** – Markdown files with step-by-step notes for Level 1 exercises:
  - `User-Accounts.md` – Managing local users
  - `Password-Management.md` – Changing passwords for users
  - `UAC.md` – Understanding and using User Account Control
- **images/** – Screenshots demonstrating the tasks

## How to Use

- Open the Markdown files in this repository to read notes and view screenshots.
- Screenshots are stored in the `images/` folder and referenced in the Markdown files.
