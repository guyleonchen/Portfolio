## Windows Fundamentals 1

**Description:**  
This lab introduced key Windows fundamentals such as user account management, password handling, and User Account Control (UAC). The exercises focused on securing Windows environments by managing accounts and enforcing best practices.  

**Skills Learned:**  
- Creating and managing local user accounts  
- Changing and securing user passwords  
- Understanding and configuring User Account Control (UAC)  

**Key Commands/Concepts:**  
- **Settings > Accounts > Family & other users** – manage local accounts  
- **Change password** – enforce secure authentication  
- **UAC (User Account Control)** – restricts administrative privileges to protect system integrity  

**Outcome:**  
Successfully documented and performed foundational Windows security tasks, including account and password management, while gaining a deeper understanding of how UAC contributes to system security.

- Repository Structure:  
  - `User-Accounts.md` – Managing local users  
  - `Password-Management.md` – Changing passwords for users  
  - `UAC.md` – Understanding and using User Account Control
