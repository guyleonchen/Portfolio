# Password Management in Windows

## Changing User Passwords

1. Go to **Settings > Accounts > Family & other users**.
2. Select the user you want to update.
3. Click **Change password**.
4. Enter a new password following Windows password requirements.

## Best Practices

- Use a **strong password** (mix of letters, numbers, symbols).  
- Change passwords regularly for security.  
- Keep passwords private and secure.
